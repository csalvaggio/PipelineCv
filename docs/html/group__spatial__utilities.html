Affine warp defined by rotation (deg), scale, and translations.

Builds a 2×3 matrix via cv::getRotationMatrix2D(center, angle_degrees, scale) about the input image center, then adds translations (tx, ty) in pixels.

Parameters
angle_degreesRotation angle in degrees (CCW).
scaleUniform scale factor (1.0 = no scaling).
txHorizontal translation in pixels (right positive).
tyVertical translation in pixels (down positive).
dsizeOutput size; (0,0) => same as input.
flagsInterpolation/warp flags (e.g. cv::INTER_LINEAR).
borderModeBorder handling (e.g. cv::BORDER_CONSTANT).
borderValueBorder value used with CONSTANT border mode.
Returns
Callable applying the specified affine transform.

Polar (or log-polar) warp from Cartesian input using cv::warpPolar.

This maps a Cartesian image to its polar representation. The x dimension of the output parameterizes angle in [0, 2π), and the y dimension parameterizes radius in [0, maxRadius].

Parameters
log_polarIf true, use logarithmic radius mapping; otherwise linear.
centerPolar center; if either component is negative, uses image center.
maxRadiusMaximum radius in pixels; if <= 0, uses farthest corner.
dsizeOutput size; if (0,0), uses input size (same rows/cols).
flagsInterpolation flags (e.g., cv::INTER_LINEAR).
Returns
Callable producing the polar/log-polar image.
